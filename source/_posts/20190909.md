####  next6.3+自带代码复制，使用代码高亮插件时hexo-prism-plugin时会不起作用

在主题下的`_config.yml`文件中设置：

{% endnote %}

```yaml
codeblock:
  # 自定义边框半径，默认是1
  # 值越大弧度越大
  border_radius: 6
  # 右上角显示复制按钮
  copy_button:
    enable: true
    # 显示复制结果
    show_result: true
```
